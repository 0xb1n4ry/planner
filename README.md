## Building and Installation	

You'll need the following dependencies:	
* libgtk-3-dev	
* libunity-dev	
* libecal1.2-dev
* libedataserver1.2-dev
* libical-dev
* libgee-0.8-dev	
* libjson-glib-dev	
* libgeoclue-2-dev	
* libsqlite3-dev	
* libsoup2.4-dev
* libgranite-dev (>=0.5)	
* libnotify-dev	
* meson	
* valac >= 0.40.3
